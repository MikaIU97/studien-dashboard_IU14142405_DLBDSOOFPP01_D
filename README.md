# studien-dashboard_IU14142405_DLBDSOOFPP01_D
## Projektbeschreibung

Das Studien-Dashboard ist eine Python-Konsolenanwendung zur Verwaltung und Darstellung studienbezogener Informationen.

Die Anwendung zeigt:

- Studienfortschritt
- Ziel-Notenschnitt
- ECTS-Fortschritt
- aktuelle Module
- Prüfungsergebnisse
- Studieninformationen

Die Daten werden aus einer JSON-Datei geladen und über eine objektorientierte Architektur verarbeitet.
